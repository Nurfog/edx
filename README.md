
edX Platform nhánh Việt Nam
=============================
Nền tảng edX Platform nhánh Việt Nam bao gồm: Hệ thống quản lý học tập dành cho người học (LMS) và Hệ thống quản lý các khóa học dành cho người dạy (Studio).
